# Woodle Lab

> Building tomorrow's tools, today.

Landing page temporaire de [Woodle Lab](https://woodlelab.com) — studio créatif et technologique.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion

## Développement

```bash
npm install
npm run dev
```

## Structure

```
src/
├── app/              # Layout + page principale
├── components/
│   ├── sections/     # Hero, Ecosystem, LabSpotlight, Footer
│   └── ui/           # AnimatedText
```
